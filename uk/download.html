Завантажити

Щоб задовільнити потреби різних користувачів, OmegaT доступна у багатьох версіях. Вибрати найзручнішу для вас версію можна за посиланням Вибір версії.

Вибір версії OmegaT

  • Тип встановлення
  • Версія
  • Операційна система
  • JRE

Недоліки: на даний момент ця версія не підтримує плагіни, тому доступні не всі функції OmegaT.

OmegaT можна встановити на будь-яку операційну систему, що може запустити Java версії 1.6.

Оберіть, чи завантажувати OmegaT з власною версією середовища Java чи без неї (JRE). Якщо потрібна версія JRE вже встановлена у вас на Windows, оберіть "Без JRE". Якщо ні, або ви не впевнені, оберіть "з JRE".

Як правило, OmegaT працює на будь-якій версії Linux, де встановлена потрібна версія середовища Java (JRE). OmegaT вимагає JRE версії 1.6 чи більше. Щоб перевірити, яка у вас версія JRE, відкрийте консоль (вікно командного рядка) та напишіть:

$ java -version

Якщо ви не впевнені, чи маєте у своїй системі потрібну версію JRE, ви завжди можете завантажити і встановити версію OmegaT із власним JRE. Ця версія JRE буде використовуватися лише OmegaT. Вона не буде ніяким чином впливати на ті версії JRE, що вже є у вашій системі, підміняти чи видаляти їх. Це абсолютно безпечний варіант встановлення.

Самодостатня версія OmegaT із власним середовищем Java. Ця версія буде довше завантажуватись та займе більше місця на диску, але ви можете бути точно впевнені, що вона нормально запуститься. Якщо сумніваєтесь - обирайте цей варіант.

Якщо ви впевнені, що на вашому комп'ютері вже є потрібна версія середовища Java, ви можете обрати варіант без JRE.

Стандартна версія OmegaT має повноцінну інструкцію користувача (щонайменше, англійською). До того ж, її, як правило, деякий час тестували на предмет помилок. Якщо ви вперше користуєтесь OmegaT - обирайте цю версію.

Як видно із назви, ця версія завжди новіша за стандартну. Але інструкція користувача може не встигати за змінами. Також ця версія значно менше тестується, тож можливі незначні баги. Незважаючи на це, "останні" версії OmegaT, як правило, дуже надійні, і їх рекомендується використовувати всім, хто вже має досвід використання OmegaT.

Вимоги

Системні вимоги

Як правило, OmegaT працює на будь-якій системі, де встановлена або може бути встановлена підходяща версія середовища Java (JRE). OmegaT може бути завантажена із власним JRE, тож нема потреби встановлювати його окремо. Можливість встановлення OmegaT протестована на Windows (всіх версій починаючи з 98), Macintosh OS X та Linux.

Файли у форматі Microsoft Office XML можна перекладати у OmegaT без додаткової конвертації. Файли у старіших форматах MS Word, Excel та Powerpoint можна перекладати, попередньо сконвертувавши їх у відповідні сучасні формати Microsoft Office або ж у формати Office Open XML. Це можна зробити за допомогою OpenOffice.org чи LibreOffice (також вільне програмне забезпечення). Посилання на OpenOffice.org та LibreOffice можна знайти на сторінці "Додаткові ресурси для OmegaT".

Вимоги до користувача

OmegaT не те щоб важко встановити чи використовувати, але для цього потрібне бажання читати і виконувати інструкції. Якщо думка про читання інструкції викликає у вас огиду, а ваш підхід до освоєння нового програмного забезпечення - це "клацай все, що бачиш", - OmegaT не для вас.

Попередні версії

Попередні версії OmegaT все ще доступні.

Але вони більше не підтримуються. Більшості користувачів вони не потрібні. Але декому доводиться встановлювати старішу версію програми з технічних причин, наприклад, через обмеження, пов'язані з операційною системою чи середовищем Java.

З/Без JRE

Для роботи OmegaT необхідно, щоб у вас на комп'ютері було встановлене середовище Java - Java Runtime Environment (JRE).

Якщо ви впевнені, що на вашому комп'ютері вже є потрібна версія середовища Java, ви можете обрати варіант без JRE. Завантаження пройде швидше, і програма займе менше місця на диску. У цьому випадку не обирайте дефолтну версію - оберіть іншу версію (наприклад, стандартну чи останню), а потім оберіть версію, позначену як "без JRE".

Якщо ви встановите версію OmegaT із JRE, то навіть якщо у вас вже було середовище Java, нова його версія встановиться окремо і ніяк не впливатиме на стару чи на програми, що її використовували. Тому можна завжди сміливо обирати версію OmegaT із JRE. Якщо ви не впевнені, чи є у вас потрібне JRE, обирайте версію із JRE (або дефолтну версію).

Користувачам Linux: більшість дистрибутивів Linux вже мають середовище Java, але деякі його відкриті імплементації погано працюють із OmegaT (це може бути помітно не одразу). Якщо ви відносний новачок у Linux, ми радимо обрати версію із JRE. Більш досвідчені користувачі Linux можуть надати перевагу встановленню версії JRE від Oracle (раніше Sun) на рівні системи.

Підписана та непідписана версії

Mac OS X має сервіс Gatekeeper, що створює перешкоди для запуску непідписаних програм. OmegaT для Mac OS X доступна як у підписаному, так і в непідписаному вигляді. Функціонально вони ідентичні, але підписану версію система розпізнає як таку, що була опублікована відомим розробником.

Підписані версії можуть виходити із затримкою. Якщо часу чекати немає, то можете використати і непідписану версію. Виникли труднощі із запуском програми? Подивіться як запустити програму від невідомого розробника.

Версія Web Start - це альтернативний спосіб встановлення OmegaT.

Оновлення для цієї версії здіснюватимуться автоматично.

Під час встановлення ви можете отримати кілька попереджень від системи безпеки вашої операційної системи.
Сертифікат підписаний компанією PnS Concept Sarl, що зареєстрована у Франції.

Для встановлення цієї версії ваша система може просити у вас "необмеженого доступу до комп'ютера". Але мається на увазі той самий дозвіл, що і для встановлення звичайної версії OmegaT, тобто доступ до жорстокого диску.

Для встановлення версії Web Start version у вас вже має бути середовище Java версії 1.3 чи більше.

Обробка персональних даних: OmegaT Java Web Start не надсилає ніяку вашу інформацію на сервери OmegaT. Програма працює на вашому комп'ютері. Ваші документи та ваша памʼять перекладів залишаються у вас на комп'ютері. Проєкт OmegaT не має доступу до вашої роботи чи ваших даних.

Версія з гілки "master" (альфа-версія) відображує останній стан розвитку програми. Вона потрібна лише розробникам та тестувальникам. Її також можна отримати за допомогою команди git:

git clone https://git.code.sf.net/p/omegat/code omegat-code

Щоденні збірки доступні на Sourceforge.

Користувачам не потрібні пакети вихідного коду. Вони призначені лише для розробників. Їх можна знайти у внутрішніх директоріях відповідних скомпільованих версій.
